# 2025.3.9-week10-解码拖延症和发音问题

## Procrastination Decoded 核心理念

### 高动机是暂时状态
- High motivation is almost by definition a temporary state
- 不能依赖动机推动行动，要靠日常小习惯
- 状态不好时也要做10分钟，保持语言链条不断

### 行动与信念的三层关系
1. **思想层**：积极还是消极的信念
2. **行为层**：习惯、工作、自我对话
3. **结果层**：成功还是失败

### 重要原则
- 每个想法要么是投资，要么是成本
- 没有想法是免费存在于头脑中的
- 选择明智的信念（it is imperative that...）

### 过程导向思维
- 对结果静待花开，对行动迫不及待
- 幸福不依赖于成功，过程本身就有价值
- 小努力日复一日累积（day in and day out）

## 发音问题解决方案

### 诊断方法
- 让学习者朗读一段话
- 观察哪些词卡壳、哪些音发不准

### 常见问题
- 长难词拼读困难 → 系统过一遍音标
- 长短元音分不清 → 元音要发饱满
- 听词反应不过来 → 词汇的声音积累太少

### 解决步骤
1. 花2小时系统过48个音标
2. 重点过元音，长元音要长、短元音要短
3. 每天在欧路词典刷词时听发音
4. 时间够的话做连读练习

## 学习目标设定

### 以周为单位
- 设想一周后你希望达成什么
- 用那个目标来反推现在要做什么
- 分阅读、听力、词汇、日记四个板块

### 弹性执行
- 状态好时四件事都做
- 状态不好时至少打卡一件事
- 保持语言链条不断，技能不被冷却

## 雅思备考建议

### 时间规划
- 词汇量需大于9000
- 阅读速度需达到官方标准
- 听力能理解60%以上算及格

### 口语问题
- 逻辑混乱时不要追求复杂句式
- 先用中文理清思路再表达
- 连接词准备不足就问AI提供清单
