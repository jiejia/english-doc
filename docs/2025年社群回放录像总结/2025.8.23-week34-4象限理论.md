# 2025.8.23-week34-4象限理论

## 书籍阅读法（Lean Learning）

### 带问题阅读
- 看目录，猜测章节内容
- 每个章节问自己：
  1. 作者想说什么？
  2. 凭什么这样说？
  3. 解决方案是什么？

### 章节要点
1. **Inspiration Overload** - 选择性好奇
2. **Choosing Action Over Information** - 行动优于信息
3. **Voluntary Force Functions** - 自愿的强制执行

## 四象限理论（Paul Nation）

### 象限一：Meaning-Focused Input（聚焦意义的输入）
- 阅读英文原著、看电影电视剧
- 关注内容含义，不纠结语法
- 生词率需低于10%

### 象限二：Meaning-Focused Output（聚焦意义的输出）
- 写英文日记、口语话题练习
- 重点：把意思表达出来
- 建立在输入基础之上

### 象限三：Language-Focused Learning（语言知识学习）
- 语法、词汇、发音的刻意学习
- 中国学生最熟悉的板块
- 应占总时间的1/4

### 象限四：Fluency Development（流利度发展）
- 重复练习已掌握的材料
- 从"能说"到"说得好"
- 心理难度最高，但必须做

## 四象限的五个前提条件

1. **Familiar** - 材料是熟悉的
2. **Interesting** - 对材料感兴趣
3. **Low difficulty** - 生词率低于10%
4. **Breadth or Depth** - 广度或深度至少满足一个
5. **Quantity** - 量要足够大

## 时间分配建议

### 比例原则
- 学占1/4，练占3/4
- 四个板块长期需要平衡
- 短期可有侧重，长期要均衡

### 难度排序
1. Meaning-Focused Input（最简单）
2. Meaning-Focused Output
3. Language-Focused Learning
4. Fluency Development（最困难）

### 流利度练习要点
- 必须练习已经练过的材料
- 重复练习同一话题3-5遍
- 目标：不假思索说出3-5句话
