# 2025.6.28-week26-i+1理论

## 克拉申五大假说

### 1. 习得与学得假说（Acquisition vs Learning）
- **学得**：通过分析语法规则，有意识地学会（约占30%）
- **习得**：在可理解输入中无意识获得，水到渠成

### 2. 监控假说（Monitor Hypothesis）
- 头脑中有个"语法警察"监控输出
- 三个条件同时满足才会触发：
  1. 有足够时间
  2. 注重语法正确性
  3. 具备语法知识
- 有人监控严（说不出口），有人监控松（能说但错多）

### 3. 自然顺序假说（Natural Order Hypothesis）
- 某些语法点天生更早掌握
- 有些语法在当前阶段强求也学不会
- **启示**：不要拔苗助长，顺其自然

### 4. 输入假说（Input Hypothesis）
- **i+1理论**：输入略高于当前水平才有效
- 三个条件：
  1. 可理解
  2. 包含略超当前水平的语法
  3. 来自自然的语言使用

### 5. 情感过滤假说（Affective Filter Hypothesis）
- 焦虑会阻挡语言进入大脑
- 保持低焦虑、高动机的学习环境
- 心情平和时学习效果最好

## Mindy的i-1理论

### 为什么推荐i-1
- 大多数人不清楚自己的真实水平i
- 与其猜测i+1，不如找确定能懂的i-1
- i-1让阅读流畅，建立信心和习惯

### i-1操作建议
- 前30小时：选择70-80%能理解的材料
- 30-60小时：逐步提升难度
- 选择标准：不查词也能抓住大意

## 选材四原则

1. **可理解性** - 至少理解50-60%以上
2. **关联性** - 与个人兴趣相关
3. **有趣性** - 让学习过程愉快
4. **重复性** - 窄读窄听，同一主题/作者反复接触

## 输出的N+1

### 从输出层面升级表达
- 先说出自己能说的版本（如：I like coffee）
- 再寻找更丰富的表达（如：I'm really into coffee）
- 在已有基础上逐步升级

### 10遍输入=1遍输出
- 一个词至少遇见10次，才可能自然产出1次
- 产出后印象更深刻
- 不要期待一次就用好
