# 2025.1.11-week2-阅读非虚构类书籍步骤

## 非虚构类书籍阅读策略

### 与虚构类的区别
- **虚构类**：只用文档模式阅读，核心是读完和保持好的阅读体验
- **非虚构类**：需要使用脑图模式，不仅读懂文字，还要运用书中原理指导生活

### 阅读流程
1. 先浏览目录，预判章节内容
2. 阅读时将有感触的句子标记到脑图中
3. 第二遍只看脑图中标记的精华内容
4. 必要时导出Word文档便于复习

## 语料库积累方法

### 查词习惯
- 使用欧路词典跨软件取词
- 录入的不是单词，而是**词组搭配**
- 例如：dispel the illusion、to the detriment of、relinquish control

### 播客听力输入
- 选择感兴趣的主持人和主题进行大量泛听
- 一期节目可以反复听2-3遍
- 先盲听了解大意，再认真听一遍做笔记
- 长篇内容不做精听，做有质量的泛听即可

## 学习时间管理

### 以周为单位规划
- 不要以年为维度思考，容易产生不切实际的幻想
- 每周复盘学习进度，根据实际情况调整
- 普通人一年真正能分给英语的时间可能只有200-300小时

### iPad工具使用
- Margin Note 3 用于阅读和标注
- 欧路词典用于词汇收集
- 两个APP分屏使用，边读边收录
