# 2025.4.26-week17-designingyourlife

## 核心阅读策略

### 两遍阅读法
1. **第一遍：把书读薄** - 关注主旨大意，不纠结每个生词，快速获取核心信息
2. **第二遍：把书读厚** - 深入分析语言点，收集有价值的词组和表达

### 词汇积累方法
- 使用欧路词典收集词组搭配（collocation），而非单个单词
- 关注"小词"的精妙用法，如 "call off" 替代 "cancel"
- 记录词汇收集数量，用数据追踪学习进度
- 采用30%原则：每页精选30%的词组进行收藏

## 阅读理解技巧

### 中英文语序差异
- 英文重点前置：主干信息放在句首，修饰成分后置
- 中文循序渐进：按逻辑顺序层层推进
- 阅读时先抓主干，再理解修饰部分

### 生词处理策略
- 不要逐词查询，通过上下文猜测词义
- 避免回读（re-reading），保持阅读流畅性
- 把不懂的内容暂时放下，让大脑自然处理

## 学习心态

### 长期主义视角
- 英语学习没有秘诀，只有坚持
- 根据个人情况调整学习强度，但保持学习的"火苗"
- 阅读量是提升的关键：量变引起质变

### 输出练习
- 读完一本书后，用3个要点总结所学内容
- 尝试用英文表达自己的理解和感悟
- 输出是检验输入效果的最佳方式
