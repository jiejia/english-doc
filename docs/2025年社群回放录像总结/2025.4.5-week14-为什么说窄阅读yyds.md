# 2025.4.5-week14-为什么说窄阅读yyds

## 窄阅读（Narrow Reading）的价值

### 什么是窄阅读
- 在某一阶段只读一个作者的书籍
- 或只读同一个话题的内容
- 语料重复次数更多，效果更好

### 核心研究结论
- 至少遇到12次才能可靠习得词汇
- 需要知道9000词家族才能理解98%的文本
- 词汇丰富度提升带动整体熟练度

### 窄阅读的好处
1. **高度激励**：读着读着速度自然加快
2. **自信迁移**：阅读自信能迁移到听力/写作/口语
3. **不经意习得**：读者无意识中吸收含义
4. **系列书hook住注意力**：一套书勾住兴趣不放

## 阅读材料选择建议

### 系列书籍推荐
- 纳尼亚传奇（难度适中）
- 沙丘系列
- 哈利波特系列
- 迪士尼系列电影原著

### 什么时候加外刊
- 完整读完10本英文书之后
- 输入技能发展成熟后
- 外刊信息浓度高，需要聚精会神

## 教辅书籍使用方法

### 雅思写作口语词组书
1. 只看英文部分，理解含义
2. 反问自己：这句话我能写出来吗？
3. 对比差距，发现表达缺口
4. 收录到欧路词典

### 不追求100%掌握
- 学了100个词组，争取用上10%
- 余下的作为语料储存
- 需要时再提取，提取不出来再查漏补缺

## 窄输入（Narrow Input）延伸

### 阅读+听力同源
- 读什么就听什么
- 读完一个章节后听有声书版本
- 激活语音记忆，不是精听，是磨耳朵

### 学习进度记录
- 每天在群里总结当日进度
- 过去记录是不放弃的动力
- 抵抗重复的力量：知道尝试了多少次

## 浸透式学习

- 真正扎扎实实跟语言打交道
- 不要在假大空方法论里浸透
- 让英语浸透自己，效果才会好
